Re-combinatorial vs. Generative

Recombinatorial music is making combinations of segmented tracks designed for vertical and horizontal organization into random containers

Generative music is where the random containers have melodic constraints (For example, chords), and rhythms and notes are generated, utilising Pseudo-Random Number Generation (PRNG).

Generative + Algorithmic music creation:

Algorithms for harmony, dynamics, melodic shapes, rhythmic patterns; these interactive algorithms are then used as constraints for generative music.

Generative Grammars

Use language-based techniques to create a music-theoretic language that will be used for making music.

Markov Models

You can use interpolation procedures to make branches - multiple different transitions to create different, similarly styled pieces.
