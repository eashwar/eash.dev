Sets

A set is a collection of items.
One notation is the curly braces.
For example:

  • {a,n,r,b,o} is the set of characters needed to spell “Ann Arbor”
  • {mon,wed,fri,sat,sun} is the set of days where EECS 203 does not have lectures.

Clarifications:

  1. A set is unordered
  2. An item appears in the set or it doesn’t. No such thing as multiple appearances.
  3. A set can contain other sets.
  4. A set can have an infinite number of items.
  5. A set can also contain nothing ({} = )

The formal definition: A set is defined by a membership predicate, i.e. x S means “x exists in the set S”
